# Обновление сайта Glow-Blocks2 на GitHub Pages

Сайт: https://egorbarsik2-svg.github.io/Glow-Blocks2/

Репозиторий обычно находится здесь:

https://github.com/egorbarsik2-svg/Glow-Blocks2

## Что загрузить

Замените файлы в корне репозитория `Glow-Blocks2`:

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

Файл `_headers` нужен для Netlify. GitHub Pages его не использует, но он не мешает.

## Как обновить через сайт GitHub

1. Откройте репозиторий `Glow-Blocks2`.
2. Нажмите `Add file` -> `Upload files`.
3. Перетащите обновленные файлы из папки `block-blast-offline`.
4. Подтвердите замену файлов.
5. Нажмите `Commit changes`.
6. Подождите 1-3 минуты.
7. Откройте сайт и нажмите `Ctrl + F5`:
   https://egorbarsik2-svg.github.io/Glow-Blocks2/
