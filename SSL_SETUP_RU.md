# Как получить HTTPS / SSL для игры

SSL-сертификат нельзя добавить внутрь `index.html`. Его выдает хостинг, где опубликована игра.

## Самый простой способ

1. Откройте Netlify Drop: https://app.netlify.com/drop
2. Перетащите туда всю папку `block-blast-offline`.
3. Netlify выдаст ссылку вида `https://name.netlify.app`.
4. Откройте эту ссылку. Если адрес начинается с `https://`, SSL уже работает.
5. В Google Sites вставьте именно эту ссылку: `Вставка -> Встроить -> URL`.

## Важно

- `file:///C:/.../index.html` не будет считаться защищенным сайтом.
- `http://...` не будет считаться защищенным сайтом.
- Нужен именно публичный адрес `https://...`.
- Файл `_headers` уже добавлен в проект. Netlify использует его для безопасных заголовков.
